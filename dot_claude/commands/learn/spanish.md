# /learn:spanish - Урок испанского языка

Генерация ежедневного урока испанского языка с сохранением прогресса в Obsidian vault.

## Triggers
- Ежедневное изучение испанского
- Продолжение курса с текущего урока
- Повторение пройденного материала

## Usage
```
/learn:spanish [--lesson N] [--review] [--status]
```

**Аргументы:**
- `--lesson N` — открыть конкретный урок (по умолчанию: следующий)
- `--review` — повторение последних 5 уроков
- `--status` — показать текущий прогресс

## Программа курса

| Уровень | Уроки | Фокус |
|---------|-------|-------|
| **A1** | 1-90 | Основы: алфавит, числа, приветствия, Presente, базовая лексика (500 слов) |
| **A2** | 91-180 | Pretérito, Imperfecto, Futuro, повседневные ситуации (1500 слов) |
| **B1** | 181-270 | Subjuntivo, условные, косвенная речь, дискуссии (3000 слов) |
| **B2** | 271-365 | Сложные времена, идиомы, профессиональный язык (4500+ слов) |

## Behavioral Flow

Когда пользователь вызывает `/learn:spanish`:

1. **Чтение прогресса**
   ```bash
   Read "20 Projects/Study/Espanol/Spanish Progress.md"
   ```
   Извлечь:
   - `Текущий урок` — номер последнего пройденного урока
   - `Текущий уровень` — A1/A2/B1/B2
   - `Последний урок` — дата

2. **Определение урока**
   ```
   Если --lesson N → использовать N
   Если --review → показать уроки (current-4) до current
   Если --status → показать прогресс и выйти
   Иначе → следующий урок = текущий + 1
   ```

3. **Проверка существования урока**
   ```bash
   ls "20 Projects/Study/Espanol/Lessons/Lesson {N}.md"
   ```

4. **Если урок НЕ существует — генерировать:**

   Определить тему по номеру урока из учебного плана (см. ниже).

   Создать урок с компонентами:

   ### Структура урока (30+ мин)

   ```markdown
   # Урок {N} — {Тема}

   ## Тема урока
   Краткое введение в тему на русском

   ## Новая лексика (10-15 слов)
   | Español | Русский | Пример |
   |---------|---------|--------|
   | palabra | слово | Ejemplo de uso |

   ## Грамматика
   ### {Грамматическая тема}
   - Правило с объяснением
   - Таблица спряжения (если применимо)
   - 5+ примеров с переводом

   ## Диалог
   Реалистичный диалог (8-12 реплик) на тему урока
   с переводом ключевых фраз

   ## Упражнения
   ### 1. Переведите на испанский (5 предложений)
   ### 2. Заполните пропуски (5 предложений)
   ### 3. Составьте предложения из слов (5 заданий)

   ## Культурная заметка
   Интересный факт об испаноязычных странах

   ## Ответы к упражнениям
   (в скрытом блоке <details>)

   ---
   → См. также: [[Spanish Grammar]], [[Spanish Vocabulary]]
   ← [[Lesson {N-1}]] | [[Lesson {N+1}]] →
   ```

5. **Сохранить урок**
   ```bash
   Write "20 Projects/Study/Espanol/Lessons/Lesson {N}.md"
   ```

6. **Обновить прогресс**
   В файле `Spanish Progress.md` обновить:
   - `Текущий урок` → N
   - `Текущий уровень` → по номеру урока (A1: 1-90, A2: 91-180, B1: 181-270, B2: 271-365)
   - `Последний урок` → сегодняшняя дата
   - `Начало обучения` → если первый урок, записать дату
   - `Дней подряд` → если последний урок был вчера, увеличить на 1; иначе сбросить в 1
   - `Всего слов изучено` → добавить количество новых слов из урока
   - `Грамматических тем` → увеличить если урок содержит новую грамматику
   - Обновить прогресс-бары (процент от уроков в текущем уровне)

7. **Обновить Spanish Vocabulary.md**
   Добавить новую секцию с лексикой урока:
   ```markdown
   ## Урок {N}: {Тема}

   | Español | Русский | Пример |
   |---------|---------|--------|
   | palabra | слово | Ejemplo |
   ```
   - Добавлять в конец файла перед `## Связанные файлы`
   - Обновить счётчик слов в статистике файла

8. **Обновить Spanish Grammar.md** (если урок содержит новую грамматику)
   Добавить или дополнить секцию грамматики:
   - Если тема уже есть — дополнить примерами
   - Если новая тема — создать секцию

   Примеры тем для добавления:
   - Уроки 31-40: Секция "Ser vs Estar"
   - Уроки 41-50: Секция "Presente Indicativo"
   - Уроки 91-100: Секция "Pretérito Indefinido"
   - и т.д. по учебному плану

10. **Вывести урок пользователю**
    Показать полный текст урока в чате

11. **Git commit**
    ```bash
    git add "20 Projects/Study/Espanol/"
    git commit -m "Complete Spanish lesson {N}: {Тема}

    - Added {X} new words to vocabulary
    - Updated grammar: {тема грамматики если есть}"
    git push
    ```

## Учебный план (365 уроков)

### A1: Уроки 1-90

**Месяц 1 (1-30): Основы**
- 1-5: Алфавит и произношение
- 6-10: Числа 1-100
- 11-15: Приветствия и прощания
- 16-20: Представление себя
- 21-25: Семья и родственники
- 26-30: Артикли (el/la/los/las, un/una)

**Месяц 2 (31-60): Presente**
- 31-40: Глаголы ser и estar
- 41-50: Presente indicativo (-ar, -er, -ir)
- 51-55: Неправильные глаголы (ir, tener, hacer)
- 56-60: Вопросы и отрицания

**Месяц 3 (61-90): Практика**
- 61-70: Еда и напитки, в ресторане
- 71-80: Покупки и магазины
- 81-85: Описание внешности и характера
- 86-90: Время, дни недели, месяцы

### A2: Уроки 91-180

**Месяц 4 (91-120): Прошедшее**
- 91-100: Pretérito Indefinido (правильные)
- 101-110: Pretérito Indefinido (неправильные)
- 111-120: Pretérito Imperfecto

**Месяц 5 (121-150): Будущее и сравнения**
- 121-130: Futuro Simple
- 131-140: Ir a + infinitivo
- 141-150: Сравнительные конструкции

**Месяц 6 (151-180): Повседневность**
- 151-160: Путешествия и транспорт
- 161-170: Здоровье и части тела
- 171-180: Погода и природа

### B1: Уроки 181-270

**Месяц 7 (181-210): Subjuntivo**
- 181-195: Presente de Subjuntivo
- 196-210: Uso del Subjuntivo (deseo, duda, emoción)

**Месяц 8 (211-240): Условные**
- 211-225: Condicional Simple
- 226-240: Oraciones condicionales (Si...)

**Месяц 9 (241-270): Косвенная речь**
- 241-255: Estilo indirecto
- 256-270: Expresar opinión, argumentar

### B2: Уроки 271-365

**Месяц 10 (271-300): Сложные времена**
- 271-285: Pretérito Perfecto de Subjuntivo
- 286-300: Pretérito Pluscuamperfecto de Subjuntivo

**Месяц 11 (301-330): Идиомы**
- 301-315: Modismos y expresiones
- 316-330: Registro formal vs informal

**Месяц 12 (331-365): Мастерство**
- 331-345: Variantes del español (España, México, Argentina)
- 346-360: Vocabulario profesional
- 361-365: Repaso general y evaluación B2

## Примеры использования

### Следующий урок
```
/learn:spanish
```
Генерирует следующий урок по порядку.

### Конкретный урок
```
/learn:spanish --lesson 42
```
Открывает или генерирует урок №42.

### Проверить прогресс
```
/learn:spanish --status
```
Показывает текущий прогресс без генерации урока.

### Повторение
```
/learn:spanish --review
```
Показывает список последних 5 уроков для повторения.

## Хранение данных

```
20 Projects/Study/Espanol/
├── Spanish Progress.md      # Прогресс и статистика
├── Spanish Vocabulary.md    # Словарь (обновляется каждый урок)
├── Spanish Grammar.md       # Грамматика (обновляется при новых темах)
└── Lessons/
    ├── Lesson 001.md
    ├── Lesson 002.md
    └── ...
```

### Связи между файлами

- Каждый урок содержит ссылки: `→ См. также: [[Spanish Grammar]], [[Spanish Vocabulary]]`
- Vocabulary группируется по урокам с заголовками `## Урок N: Тема`
- Grammar структурирована по темам с примерами из уроков

## Обработка ошибок

| Ситуация | Действие |
|----------|----------|
| Spanish Progress.md не существует | Создать с начальными значениями (урок 0, уровень A1) |
| Папка Lessons/ не существует | Создать `mkdir -p "20 Projects/Study/Espanol/Lessons/"` |
| Урок уже существует | Показать существующий урок, не перезаписывать |
| Запрошенный урок > 365 | Показать сообщение о завершении курса |

## Советы

- Проговаривай диалоги вслух
- Делай упражнения письменно
- Используй `--review` раз в неделю
